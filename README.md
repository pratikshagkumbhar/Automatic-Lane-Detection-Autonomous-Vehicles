# Automatic-Lane-Detection-Autonomous-Vehicles
Automatic lane detection system for autonomous vehicles developed in MATLAB. Uses Gaussian filtering, Canny edge detection, ROI selection, and Hough Transform to detect road lanes from video frames. Includes lane tracking, smoothing, and basic turn prediction for intelligent driving assistance.

## Features

* Grayscale conversion
* Gaussian filtering
* CLAHE enhancement
* Canny edge detection
* ROI extraction
* Hough Transform lane detection
* Lane tracking and smoothing
* Turn prediction (Left/Right/Straight)
* MATLAB GUI implementation

## Methodology

Input Video → Preprocessing → Edge Detection → ROI Selection → Hough Transform → Lane Detection → Lane Smoothing → Turn Prediction
